Wireless - Access Control
Access Control allows you to block the access from certain wireless stations or to bypass access from certain wireless stations only. In Accept mode, WL300g will only accept wireless access from stations with MAC address in the control list. In Reject mode, WL300g will reject wireless access from stations with MAC address in the control list.
